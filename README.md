fetch_data.py will fetch all the necessary  data that you require. 

Once you have that, run features.py to compute the reqd features, and weekly.py to aggregate  them as weekly as-of features. 

Run threshold_model.py to run the regime detector. You can play with the parameters in the data class or apply an ML layer to optimize them. 

plot_risk.py and plot_structure.py will give a visual result of the regimes. 

Reports contains the project report and plots.
